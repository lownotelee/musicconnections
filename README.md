This is a map showing the connections between different bands and musicians. I have been fascinated for a long time as to how different bands relate to each other, and how those connections span decades. One example I love is how Dave Grohl of Nirvana plays in Them Crooked Vultures, which also features John Paul Jones of Led Zeppelin.

This aims to show those links graphically, and hopefully introduce people to some new music that their favourite musicians or bands have crossed paths with.
